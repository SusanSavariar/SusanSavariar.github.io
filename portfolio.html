CheatSheet- HTML
January 10th, 2016
CSS stands for Cascading Style Sheets and describes how HTML elements are to be displayed on screen or paper. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files. A CSS rule set consists of a selector and a declaration block.The selector points to the HTML element you want to style.The declaration block contains one or more declarations separated by semicolons.Each declaration includes a CSS property name and a value, separated by a colon.A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.
